DIREÇÃO VISUAL DE CARROSSEL — INOVA FORTIS EPIs

A direção visual dos carrosséis deve ser forte, organizada e pensada para retenção.

O design não é apenas estética.

Ele é parte da comunicação.

FORMATO OBRIGATÓRIO

Todos os slides devem ser produzidos no formato:

1080 x 1350 px — proporção 4:5

Esse é o formato padrão de carrossel do Instagram.

Não utilizar outros formatos ou proporções.

REGRA CRÍTICA DA CAPA

A capa deve ser entregue exclusivamente como um prompt completo em inglês para geração de imagem em IA.

Não deve ser entregue como layout, descrição ou texto separado.

Deve ser um prompt pronto para uso, contendo:

descrição detalhada da cena
ambiente realista e profissional
personagem coerente com o tema
iluminação cinematográfica realista
composição visual completa
cores da marca aplicadas
tipografia especificada
texto em português do Brasil já inserido
posicionamento do texto
linha laranja no rodapé
logo da Fortis visível e bem posicionado

A capa é gerada fora do Claude e deve vir pronta para uso.

OBJETIVO DA CAPA

Parar o scroll imediatamente.

Gerar impacto visual.

Criar curiosidade.

Representar fielmente o conteúdo.

REGRA DE VARIAÇÃO VISUAL

Os carrosséis não devem seguir sempre o mesmo estilo.

Cada carrossel deve ter identidade visual própria.

O feed não pode parecer repetitivo.

VARIAÇÃO OBRIGATÓRIA ENTRE CARROSSÉIS

Variar entre carrosséis:

paleta de cores dominante (dentro das cores da marca)
estilo tipográfico e hierarquia
composição geral dos slides
ritmo visual — denso, clean, dramático, técnico
uso de elementos gráficos — blocos, linhas, ícones, formas
estrutura de cada slide — não repetir os mesmos layouts

VARIAÇÃO OBRIGATÓRIA DENTRO DO CARROSSEL

Variar entre slides do mesmo carrossel:

fundo — nunca todos os slides com o mesmo fundo
tipografia interna — tamanhos e pesos diferentes por slide
posição dos elementos — não centralizar tudo sempre
uso de cor — distribuir laranja, amarelo e azul com critério

O padrão é consistência de marca, não repetição de layout.

Slides iguais entre carrosséis diferentes são proibidos.

Repetição visual torna o feed previsível e reduz retenção e autoridade.

TIPOGRAFIA

A tipografia deve ser forte, legível e bem distribuída.

Nunca utilizar texto pequeno demais.

Nunca comprometer a leitura.

O texto deve ser facilmente legível no celular.

TAMANHOS MÍNIMOS OBRIGATÓRIOS (em HTML/CSS, base 1080px)

Títulos principais: mínimo 56px
Subtítulos e textos de destaque: mínimo 32px
Textos de corpo e descrição: mínimo 26px
Labels, tags e eyebrows: mínimo 18px
Nenhum texto visível ao usuário abaixo de 18px

Priorizar:

títulos grandes
boa hierarquia
espaçamento adequado

Evitar:

texto comprimido
fontes difíceis
excesso de informação no mesmo bloco

FUNDOS DOS SLIDES

Utilizar:

fundos escuros
cores da marca
contraste alto
texturas leves

O fundo deve destacar o texto, não competir com ele.

ELEMENTOS VISUAIS

Os slides devem conter construção visual.

Utilizar:

linhas
faixas
ícones
símbolos
gráficos simples
destaques de palavras

Esses elementos ajudam na retenção.

Evitar slides apenas com texto simples.

CORES

Aplicar cores da Fortis de forma estratégica:

azul como base
laranja para destaque
amarelo para alertas
preto para contraste

Manter equilíbrio.

Não poluir visual.

CONTRASTE E VISIBILIDADE — REGRA CRÍTICA

Todo texto deve ser plenamente legível e presente.

Proibido:

usar opacity abaixo de 0.6 em qualquer texto visível
usar cores de texto com hex abaixo de #444444 em fundos escuros
usar cores de texto acima de #bbbbbb em fundos claros
criar texto "fantasma" ou quase invisível como elemento decorativo

Textos secundários e subtextos podem ter cor mais suave, mas nunca abaixo de:

#666666 em fundo escuro
#888888 como limite absoluto em qualquer contexto

Se o texto está no slide, ele precisa ser lido com facilidade. Texto que não pode ser lido não deve existir.

IMAGENS

Quando utilizadas:

devem ser realistas
devem estar alinhadas com o conteúdo
devem representar corretamente o EPI

Evitar imagens genéricas.

REGRA DO LOGO

O logo da Fortis deve aparecer no primeiro slide obrigatoriamente.

Deve estar:

sem fundo
bem posicionado
visível
com tamanho adequado

Não pode ser pequeno demais.

Nos demais slides, pode aparecer, mas não é obrigatório.

COERÊNCIA VISUAL

Toda imagem e elemento deve estar alinhado com o conteúdo.

Se fala de um EPI, ele deve estar correto.

Se fala de um risco, ele deve ser representado corretamente.

OBJETIVO DO DESIGN

Chamar atenção.

Facilitar leitura.

Aumentar retenção.

Reforçar a mensagem.

REGRA FINAL

Se a capa não estiver em formato de prompt completo, está errado.

Se os slides parecerem todos iguais, está errado.

Se o texto não estiver legível, está errado.

Se o design não ajudar a entender, não representa a Fortis.
