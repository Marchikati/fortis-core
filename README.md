# Fortis Core

This is the central project for Inova Fortis EPIs.

It contains the complete system used to create:

- content (social media, reels, carousels)
- landing pages
- link in bio pages
- catalog materials
- campaigns
- brand assets

---

## STRUCTURE

- context/
  Core system of the brand.
  Includes positioning, communication, audience, content strategy and rules.

- central/
  Landing pages and central access pages (link in bio, QR pages).

- social/
  Content creation system (ideas, scripts, captions, campaigns).

- catalogo/
  Catalog structure, materials and product organization.

- prompts/
  Reusable prompts for Claude Code and other tools.

---

## CORE PRINCIPLE

Fortis does not exist to sell first.

It exists to teach, guide and create awareness about real risks.

Sales are a consequence of trust.

---

## SYSTEM USAGE

Before generating anything:

1. Read everything inside /context
2. Follow the brand system strictly
3. Execute the task inside the correct module

Examples:

- /central → pages and landing experiences
- /social → content and campaigns
- /catalogo → catalog development

---

## RULES

- Never generate generic content
- Never create purely promotional content
- Always deliver practical value
- Always align with Fortis positioning
- Always prioritize clarity and real-world application

---

## FINAL RULE

If it does not educate, alert or improve safety, it does not belong to Fortis.
